## Production Notes

- **Stitch (design tool):** Use for static mockup generation — see /.stitch-cards/ for existing card designs
- **Phone mockups:** Use real Waggle screenshots, not mock data
- **Brand consistency:** All assets use forest green/gold/cream palette, Playfair + Inter fonts
- **A/B testing:** Run 2-3 headline variants per static ad. Let Meta optimize after 500 impressions.
- **UTM tracking:** All ad links include `?utm_source={platform}&utm_medium=paid&utm_campaign={campaign_name}`
