# Waggle Partner One-Pager

## What Is Waggle
Waggle is the social sportsbook for golf groups. We help trip organizers, member-guest directors, and scramble hosts run live bets, scoring, and settlement from any phone with no app download.

Groups use Waggle to keep games moving and end every round with a clean payout screen instead of arguments and spreadsheet math.

## How It Works
- Host creates an event in minutes and shares one link or QR code with the group.
- Players join from their phones, post scores, and run games like Nassau, skins, wolf, and match play live.
- Waggle auto-calculates standings and settlement so everyone sees exactly who owes what.
